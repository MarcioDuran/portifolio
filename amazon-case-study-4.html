Case Study: Redesign da Experiência de Compra na Amazon

Ideação

Abordagem com Heurísticas de Nielsen

Para nossa fase de ideação, utilizamos as 10 Heurísticas de Usabilidade de Nielsen como um framework para gerar soluções que não apenas resolvessem os problemas identificados, mas também garantissem uma experiência do usuário de alta qualidade.

Sessão de Brainstorming Guiada por Heurísticas

Realizamos uma sessão de brainstorming usando a técnica do Crazy 8's, mas com um twist: cada rodada foi focada em uma das heurísticas de Nielsen. Isso nos ajudou a gerar ideias diversas e alinhadas com princípios de usabilidade estabelecidos.

Heurística de Nielsen Ideias Geradas
1. Visibilidade do status do sistema
  • Barra de progresso para o processo de checkout
  • Indicadores visuais de itens comparados/salvos
2. Correspondência entre o sistema e o mundo real
  • Categorias de produtos com ícones familiares
  • Metáfora de "provador virtual" para visualização de produtos
3. Controle e liberdade do usuário
  • Botão "Desfazer" para ações de remoção do carrinho
  • Opção de salvar carrinhos para compra posterior
4. Consistência e padrões
  • Design de cards consistente para todos os produtos
  • Padronização dos botões de ação em toda a interface
5. Prevenção de erros
  • Confirmação antes de remover itens do carrinho
  • Sugestões de correção para buscas com erros de digitação

... (continuação para as outras 5 heurísticas)

Priorização de Ideias

Após o brainstorming, utilizamos uma matriz de impacto vs. esforço para priorizar nossas ideias. As soluções de alto impacto e baixo esforço foram selecionadas para prototipagem:

  1. Redesign da navegação por categorias com ícones intuitivos
  2. Implementação de um comparador de produtos side-by-side otimizado para mobile
  3. Simplificação do processo de checkout com opção de "compra rápida"
  4. Redesign das páginas de produto com informações essenciais em primeiro plano
  5. Adição de filtros de busca mais granulares e fáceis de usar em mobile

Esboços e Wireframes Iniciais

Com base nas ideias priorizadas, criamos esboços rápidos e wireframes de baixa fidelidade para visualizar nossas soluções:

  • Esboços de mão livre para explorar rapidamente diferentes layouts
  • Wireframes digitais básicos para refinar a estrutura da interface
  • Fluxogramas simplificados para mapear a nova jornada do usuário

Esses esboços e wireframes serviram como base para nossa fase de prototipagem, permitindo-nos iterar rapidamente e refinar nossas ideias antes de investir em designs mais detalhados.
